# Tic Tac Toe (Java)

This is a simple Tic Tac Toe game developed in Java.

Features:
- Two-player mode (X and O)
- Input validation
- Win and draw detection
- Scoreboard tracking

How to Run:
1. Compile the program
2. Run ClassProject.java
3. Follow on-screen instructions
